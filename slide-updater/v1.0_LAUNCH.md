## O que é?

**Slide Updater** é um editor acadêmico para slides com:
- ✅ **Rich text editing** (WYSIWYG)
- ✅ **Literature search** (PubMed + Scopus)
- ✅ **Design analysis** (TF-IDF + recomendações)
- ✅ **Undo/Redo** (histórico ilimitado)
- ✅ **Multi-format export** (HTML, PPTX, JSON)
- ✅ **Full accessibility** (WCAG AA)

## Como Usar

### Instalação

```bash
git clone https://github.com/[user]/slide-updater.git
cd slide-updater
npm install
npm run dev
```

### Workflow Típico

1. **Criar projeto** → "+ Novo Projeto"
2. **Importar slides** → "📤 Importar Slides" (PDF/PPTX/TXT)
3. **Editar conteúdo** → RichTextEditor com formatação
4. **Buscar literatura** → "🔍 Buscar Literatura"
5. **Revisar sugestões** → "✨ Sugestões de Design"
6. **Exportar** → HTML, PPTX ou JSON backup

### Configurar Scopus (opcional)

1. Clique ⚙️ (Settings)
2. "Adicionar Chave" → Cole API key Scopus
3. Próximas buscas usarão Scopus + PubMed
